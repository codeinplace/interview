# Entrevista
CodeInPlace test interview

O objetivo desse teste é revelar o seu nível de experiência. Pessoas de diferentes níveis executarão esse teste de diferentes maneiras. Acima de tudo esperamos que você adquira experiência e se divirta programando.

Em um mercado competitivo mais importante que se especializar em algo se tornar um "problem solver". Se você possui a habilidade de resolver o problema do seu cliente independente da tecnologia utilizada, com certeza terá o seu lugar de destaque. A experiência adquirida com o tempo só mostrará em qual situação e qual tecnologia deve ser utilizada. Baseando-se nisso não pedimos uma lista gigantesca de conhecimentos para poder trabalhar conosco. Mas como todo projeto, são necesśarios alguns conhecimentos para executa-lo.

## Front-end
- O layout do teste está no figma: https://www.figma.com/file/dJGJLkbnHz57N80zHLVQ75/CodeInPlace-Interview?node-id=0%3A1

- O layout deve ser responsivo e se adaptar no mobile
- O yin-yang é um ícone do Font Awesome e deve ser utilizado como SVG
